## description
A JS seamless scroll plug-in

## install
``` javascript
npm install seamless-scroll-js --save-dev
```
## use
### react or vue
``` javascript
import "seamless-scroll-js"

const dom = document.getElementById("dom");  // Get the DOM element that needs to scroll
dom.SeamlessScroll([options]);
```
### html
``` javascript
import "seamless-scroll-js"

const dom = document.getElementById("dom");  // Get the DOM element that needs to scroll
dom.SeamlessScroll([options]);
```
